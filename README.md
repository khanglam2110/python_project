# Stock Market Spillover Analysis (VAR Model)

## 📌 Overview
This project analyze spillover effects across major industry sectors in the Vietnam stock market using a Vector AutoRegression (VAR) framework.

The objective is to understand how shocks in one sector propagate to others, providing insights into market interconnectedness and risk transmission. 

--- 

## 📊 Data
- Source: VN30 stock price data collected in Investing.com
- Period: 2020-2025
- Preprocessing:
  - Cleaned and formatted raw data in Excel
  - Calculated log returns
  - Conducted stationary tests

--- 

## ⚙️ Methodology 
- Vector AutoRegression (VAR) model to capture dynamic relationships
- Forecast Error Variance Decomposition (FEVD) to quantify spillover effects
- Computation of Total Spillover INdex

--- 

## 📈 Key Insights
- Identified significant spillover effects between sectors
- Highlighted dominant transmitters and receivers in the market
- Demonstrated cross-sector risk dependencies

--- 

## Tools Used
- Python (pandas, numpy, statsmodels)
- Excel (data structuring & preprocessing)
